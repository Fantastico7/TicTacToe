#TicTacToe
=========
Simple Spark Java app with support for Selenium testing. The App is a simple game of TicTacToe with player1 vs player2 in a Web application.


Runs on (tested on):

- ✓ Windows (8.1, x86)
- ✓ Mac OS (Mavericks)
